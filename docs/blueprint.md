# **App Name**: MITA SHAREE

## Core Features:

- User Authentication: Secure user login and registration with Firebase Authentication (Email/Password).
- Product Catalog: Display products fetched from Firebase Realtime Database with categories and banners.
- Wishlist Management: Allow users to add/remove products from their wishlist, storing data in Firebase.
- Shopping Cart: Enable users to add/remove products from their cart and manage quantities.
- Order Placement: Process orders, save delivery addresses, integrate Razorpay for online payments (dummy key for MVP), and store order details in Firebase. Clear the cart after successful order placement.
- Profile Management: Allow users to update their profile name and upload profile pictures to Firebase Storage.
- Address Autocomplete Tool: Autocomplete address suggestions using the provided pincode API.

## Style Guidelines:

- Primary color: Crimson (#DC143C) for a modern and engaging feel, inspired by the request for a 'red-jaisa' shade.
- Background color: Light pink (#F9E4E6), desaturated to 25% for softness.
- Accent color: Deep pink (#FF1493) for highlighting important elements and calls to action, complementing the primary color.
- Body and headline font: 'PT Sans' for a modern and clean aesthetic.
- Use a set of consistent icons for navigation, product actions (add to cart, wishlist), and order status.
- Implement a fixed bottom navigation bar for Home, Wishlist, Cart, and Profile screens.
- Use subtle animations for transitions between screens and product interactions.
